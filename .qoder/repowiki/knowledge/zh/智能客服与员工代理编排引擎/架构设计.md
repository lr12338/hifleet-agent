- **入口分发**：`agent.py` 中的 `build_agent` 根据 Profile（`customer_support` / `employee_assistant`）动态编译不同的 StateGraph。
- **客服链路 (Customer Support)**：采用“规划-执行-校验”模式。`customer_support_router.py` 负责意图识别、实体提取及多模态感知；通过 `_execute_customer_support_harness` 调用确定性工具链（如船舶轨迹、知识库检索），失败时降级至 LLM。
- **员工链路 (Employee Assistant)**：实现受控的 Python 代码生成与沙箱执行循环。包含 `route` -> `plan` -> `act` (LLM 生成代码) -> `check` (沙箱运行) -> `loop` (错误修复) 的状态流转。
- **安全守卫**：`customer_support_guard.py` 提供输出过滤，拦截敏感信息（API Key、内部路径）并清洗原始工具日志，确保回复符合客服规范。
- **配置管理**：`profiles.py` 管理 Agent 的技能集、工具权限及提示词模板，支持通过 Header 动态切换角色。
