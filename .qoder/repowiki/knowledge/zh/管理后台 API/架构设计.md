- 采用 FastAPI 框架，通过 `router.py` 定义以 `/admin` 为前缀的路由组，并统一注入 `verify_admin_api_key` 依赖进行身份鉴权。
- 遵循分层架构：`schemas.py` 定义请求/响应数据模型，`service.py` 封装核心业务逻辑（如代理测试运行、流式聊天、观测数据查询），`auth.py` 处理基于环境变量的 API Key 验证。
- 依赖外部模块 `observability.repository` 进行日志与会话数据的持久化查询，以及 `llm_config` 模块管理大模型配置。
- 支持流式响应（SSE）用于测试运行和 Ark 聊天接口，通过 `StreamingResponse` 实现实时数据推送。
