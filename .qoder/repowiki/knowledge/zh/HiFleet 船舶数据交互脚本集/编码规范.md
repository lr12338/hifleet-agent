- 所有脚本均遵循 CLI 模式，通过 `if __name__ == "__main__":` 块结合 `argparse` 解析命令行参数并直接打印结果。
- API 认证逻辑集中化，通过 `auth.py` 中的 `public_api_key()`, `psc_api_key()`, `ttse_key()` 按优先级从环境变量获取 Token。
- 网络请求统一使用 `urllib.request.urlopen` 并设置 `timeout=10`（部分统计接口为 15s），确保异常可控。
- 写操作脚本（如 `update_static_info.py`, `upload_position.py`）采用“显式字段映射”策略，仅将用户提供的非空参数组装进 JSON 请求体，避免覆盖云端现有数据。
