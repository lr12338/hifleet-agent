- **模块化设计**：采用“共享工具+独立入口”架构，`auth.py` 统一管理多套 API Key（Public/PSC/TTSE），`coord_utils.py` 提供经纬度格式转换，其余脚本为独立 CLI 入口。
- **轻量级实现**：完全依赖 Python 标准库（`urllib`, `json`, `argparse`），无第三方框架依赖，通过环境变量动态配置 API Base URL。
- **接口边界**：各脚本封装了特定的 HiFleet API 端点（如 `/position/get/token`, `/shiparchive/getShipArchiveWithEnginAndCompany`），统一返回原始 JSON 或文本响应，由调用方进一步处理。
