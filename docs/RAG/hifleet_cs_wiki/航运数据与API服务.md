HiFleet以全球船舶、港口、海图和气象数据为核心，向客户提供数据接入、API接口、数据分析和定制化解决方案。

## 核心能力
HiFleet官网和产品介绍都强调，平台不仅提供船位实时AIS、历史AIS、历史航次、靠离港、船舶档案、全球气象、台风、潮汐、海盗事件、航行警告等数据，还把这些数据封装成 API、SDK 和行业解决方案。

面向企业用户时，HiFleet的定位不是单一查询网站，而是“数据接入平台+分析服务平台”。官网页面把能力分成“航运数据、数据分析、解决方案、报告与评估”四层，说明既能卖原始数据，也能卖模型、接口和第三方分析报告。

## 常见数据项
- 船位实时AIS数据、历史AIS数据、历史航次数据、历史靠离港记录
- 船舶档案、全球港口数据、全球电子海图
- 全球气象、台风、潮汐、地震、海盗事件、航行警告
- 船舶靠离港提醒API、船舶动态提醒API、船舶风险提醒API、水域智能监控API、全球两点间航距/航线API

## 对外表达建议
面向客户介绍时，可优先强调三点：
1. 数据覆盖广，既有岸基AIS也有卫星AIS和合作基站数据。
2. 数据形态多，可提供页面查询、API、SDK、分析报告和行业解决方案。
3. 能结合业务场景做定制，如航运监控、智慧港口、智慧口岸、海事法院、可视化大屏等。
