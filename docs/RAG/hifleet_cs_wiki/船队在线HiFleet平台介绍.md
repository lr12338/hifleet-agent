# 船队在线HiFleet平台介绍

## 概述
“船队在线HiFleet”是船舶安全管理大数据平台，整合全球多源航运数据资源，为全球各类航运相关用户提供多元化航运大数据应用与技术服务，助力实现更安全、及时、准确、智能的航运业务管理。

## 数据资源
- 船舶AIS数据：接入全球150多颗卫星、5000多座岸基、3500多座船基的实时数据，每日更新量超2亿条
- 融合资源：全球海图数据、气象数据、船舶档案数据、港口数据等多类大数据资源

## 核心服务
1. 基础船舶数据服务：全球实时船位跟踪、历史轨迹查询、航线预测、航线管理
2. 环境信息服务：海洋气象预报、台风潮汐查询、水域电子围栏监控
3. 智能管控服务：智能风险预警、全球压港查询、泊位搜索、港口分析
4. 增值应用服务：服务商地图、大宗货物跟踪分析、航运信息系统建设
5. 技术支持服务：航运数据API、定制化可视化建设

## 适用用户
船东、船舶管理公司、银行租赁企业、船舶代理、租船企业、引航机构、船员、物流公司、海事高校、航运研究机构等各类航运相关主体。
