# HiFleet客服问答对

## 使用原则
- 先回答用户当前问题，再补充必要背景，不一次性倾倒过多产品信息。
- 价格、权限开通、数据采购、异常核验等问题，统一采用“先解释规则，再引导联系人工”的方式。
- 资料未明确给出价格时，不输出未经证实的价格数字。

### QA01 HiFleet能提供哪些数据和接口？
- 分类：产品功能
- 标准答案：HiFleet以全球船舶AIS、历史航次、靠离港、船舶档案、港口、海图和海洋气象数据为核心，同时提供API接口、SDK、分析报告和行业解决方案。除了页面查询，也支持企业把数据接入自己的系统或大屏。
- 关联主题：[[航运数据与API服务]]
- 来源追溯：`wiki/航运数据与API服务.md`，`raw/产品文档/HiFleet服务产品介.md`，`raw/产品文档/网页抓取结果/数据-船队在线HiFleet.md`

### QA02 DTU是什么？适合什么场景？
- 分类：产品功能
- 标准答案：DTU是把船舶现有AIS信号接入互联网的方案，可实现更高频、更可控的实时船位回传。适合远洋船队、对船位时效要求高的船东或希望把实时船位接入自有系统的企业。
- 关联主题：[[DTU船载设备]]
- 来源追溯：`wiki/DTU船载设备.md`，`raw/产品文档/网页抓取结果/系统介绍-船队在线HiFleet（DTU）.md`

### QA03 HiFleet DTU和通用DTU有什么区别？
- 分类：产品功能
- 标准答案：HiFleet DTU是自研设备，强调即插即用、远程配置、可筛选发送内容和更可控的流量；通用DTU则更偏低成本和自助安装，但发送内容较固定，维护和升级更多依赖设备原厂。若客户更看重稳定性和后续服务，优先介绍HiFleet DTU。
- 关联主题：[[DTU船载设备]]
- 来源追溯：`wiki/DTU船载设备.md`，`raw/产品文档/网页抓取结果/系统介绍-船队在线HiFleet（DTU）.md`

### QA04 AIS接收基站可以合作吗？
- 分类：产品功能
- 标准答案：可以。HiFleet支持岸基、船载、卫星等多种AIS接收基站合作，只要具备安装场地、运营船舶或相关接收资源，都可以提交合作信息评估。合作方可获得专属船队账号、基站管理界面、合作数据共享等权益。
- 关联主题：[[AIS接收基站合作]]
- 来源追溯：`wiki/AIS接收基站合作.md`，`raw/产品文档/网页抓取结果/系统介绍-船队在线HiFleet（AIS接收基站）.md`

### QA05 智能视频监控系统能识别什么？
- 分类：产品功能
- 标准答案：HiFleet智能视频监控可识别船舶靠帮、漏油、火灾、电池自燃、长时间离岗、疲劳驾驶、吸烟、未穿工服、未戴安全帽、人员跌倒等行为，并通过微信、手机端和PC端进行报警与远程干预。

### QA06 船舶性能跟踪主要解决什么问题？
- 分类：产品功能
- 标准答案：性能跟踪主要帮助船东和船管公司统一日报与能效报送、自动校验数据、做油耗和速度分析、跟踪CII表现，并基于AIS和气象数据建立单船模型，用于航次执行分析和节能减排管理。

- 来源追溯：`wiki/船舶性能跟踪.md`，`raw/产品文档/HiFleet推出全新船舶性能跟踪服务.md`

### QA07 HiFleet的气象海况数据有什么特点？
- 分类：产品功能
- 标准答案：HiFleet提供全球气压、风、浪、流等海洋气象预报，并支持粒子图、船位气象、港口气象、轨迹气象和航线联动分析。它的优势不是单看天气图，而是把气象和船位、历史轨迹、计划航线、运营分析结合起来。
- 关联主题：[[全球气象海况]]
- 来源追溯：`wiki/全球气象海况.md`，`raw/产品文档/HiFleet上的全球气象海况.md`

### QA08 HiFleet航线优化和传统气象导航有什么区别？
- 分类：产品功能
- 标准答案：传统气象导航更偏安全和经验修正，而HiFleet航线优化更强调经济性、时效性和安全性的动态平衡。它会结合AIS、气象、船舶性能模型和历史航线数据，动态调整航线，并评估节能、失速和风险差异。
- 关联主题：[[航线优化与气象导航]]
- 来源追溯：`wiki/航线优化与气象导航.md`，`raw/产品文档/HiFleet航线优化和传统气象导航优势对比.md`

### QA09 气象预报邮件怎么开通和使用？
- 分类：产品功能
- 标准答案：前提是先给船舶添加计划航线。计划航线可通过手绘、上传、航程规划或邮箱登记方式建立，完成后在船舶当前页面的“航线气象”版块点击“给船舶发送气象报告”，即可向邮箱发送航次气象预报邮件。
- 关联主题：[[气象预报邮件]]、[[计划航线管理]]
- 来源追溯：`wiki/气象预报邮件.md`，`raw/产品文档/HiFleet新上线的气象预报邮件功能怎么用？免费用！(1).md`

### QA19 免费用户为什么看不到最新船位？
- 分类：常见问题
- 标准答案：免费用户看到的船位范围有限，卫星船位和D-AIS等数据默认不开放，所以可能无法看到更完整或更实时的最新船位。如需更高权限，可进一步确认当前账号版本并联系开通。
- 关联主题：[[船位数据常见问题]]、[[账号权限与历史轨迹范围]]

### QA21 MMSI和IMO正确，但船名为什么不对？
- 分类：常见问题
- 标准答案：船名错误可能是因为船舶改名后历史库尚未同步、船舶档案更新滞后，或其他数据源尚未覆盖。若用户提供具体船舶信息，可进一步转人工核查和修正。

### QA22 岸基AIS和卫星AIS有什么区别？
- 分类：常见问题
- 标准答案：岸基AIS主要覆盖沿海约30到50海里范围，卫星AIS则面向全球覆盖，适合远洋和更大范围的监控需求。两者的差异核心在覆盖范围和接收条件。
- 关联主题：[[AIS信号与更新频率]]
- 来源追溯：`wiki/AIS信号与更新频率.md`，`raw/平台指导文档/HIFLEET用户常见船位问题解答.md`

### QA23 为什么ETA时间不准或一直没更新？
- 分类：常见问题
- 标准答案：ETA和目的港信息的发送频率本来就低于船位，因此更新时间可能比船位更滞后。可以先等待一段时间，若明显异常，再联系人工客服核查。
- 关联主题：[[船位数据常见问题]]
- 来源追溯：`wiki/船位数据常见问题.md`，`raw/平台指导文档/HIFLEET综合问答结构化处理-9b24c499be.md`

### QA24 AIS信息多久更新一次？
- 分类：常见问题
- 标准答案：AIS更新频率取决于设备类型、船速、是否转向以及发送的是静态信息还是动态信息。一般来说，动态船位更新更频繁，而ETA、目的港等航次信息更新相对更慢。
- 关联主题：[[AIS信号与更新频率]]
- 来源追溯：`wiki/AIS信号与更新频率.md`，`raw/平台指导文档/HIFLEET用户常见船位问题解答.md`
