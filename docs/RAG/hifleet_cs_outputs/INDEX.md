# HiFleet智能客服输出包

## 文件清单
- [客服问答对.md](docs/RAG/hifleet_cs_outputs客服问答对.md)：26条高频问答，适合人工客服、知识运营和训练集初稿。
- [标准回复话术.md](docs/RAG/hifleet_cs_outputs标准回复话术.md)：13类场景化回复模板，强调先解释、再引导、必要时转人工。
- [FAQ检索词.md](docs/RAG/hifleet_cs_outputsFAQ检索词.md)：28个FAQ意图及常见搜索词、同义表达、优先命中文档。
- [客服知识库结构化.jsonl](docs/RAG/hifleet_cs_outputs客服知识库结构化.jsonl)：12条结构化样本，适合后续做检索增强、FAQ召回或训练数据扩充。

## 使用建议
- 面向人工客服：优先使用“客服问答对”和“标准回复话术”。
- 面向检索系统：优先使用“FAQ检索词”和 `jsonl` 中的 `keywords`、`related_topics`、`sources`。
- 面向后续训练：可将“客服问答对”继续拆成多轮问答、拒答规则、转人工规则和意图标签。
