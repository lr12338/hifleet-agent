## 🎯 导入策略

### 问答对处理策略

**识别规则**:
```regex
###\s+([Q\d]+|业务\d+)\s+(.+?)\n
\*\*问题\*\*[：:]\s*(.+?)\n
\*\*解答\*\*[：:]\s*(.+?)
```

**转换格式**:
```
【分类标签】问题标题

问题：具体问题内容

答案：具体答案内容
```

**分块配置**:
- 分隔符: `\n\n`
- 最大Token: 800
- 特点: 保持问答完整性，便于精确匹配

### 操作指南处理策略

**识别规则**:
```regex
##\s+([\d.]+)\s+(.+?)\n(.+?)
```

**转换格式**:
```
【操作指南】章节ID 标题

详细操作内容...
```

**分块配置**:
- 分隔符: `\n\n`
- 最大Token: 1200
- 特点: 保留操作步骤完整性

## 📝 使用示例

### Python SDK调用

```python
from coze_coding_dev_sdk import KnowledgeClient, Config

client = KnowledgeClient(config=Config())
response = client.search(
    query="如何修改密码",
    top_k=3,
    min_score=0.7
)

for chunk in response.chunks:
    print(f"相关度: {chunk.score:.2%}")
    print(f"内容: {chunk.content}")
```

### Agent自动调用

```python
from agents.agent import build_agent

agent = build_agent()
result = agent.invoke({
    "messages": [{
        "role": "user",
        "content": "岸基AIS和卫星AIS有什么区别？"
    }]
})
```
