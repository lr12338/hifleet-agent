# Skill: Browser Verify

Public web verification helper for customer_support.

Use only for public pages. Do not access authenticated pages, cookies, internal
URLs, or browser/session logs. Prefer HiFleet official domains and community
pages before public web sources.

When the caller has no target URL but the prior knowledge/web search result is
weak and the question may belong to HiFleet product/community/help content, use
the query keywords to search Bing. Rank HiFleet official domains, community
posts, help center pages, and official site pages ahead of public fallbacks.
