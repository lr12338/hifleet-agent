# Skill: knowledge-qa

## 概述
知识检索与问答域，处理所有信息查询请求。统一使用smart_search工具，内部自动5层路由。

## 适用场景
- Hifleet平台功能咨询（如何注册、怎么用DTU、船位问题等）
- 航运行业知识（AIS原理、海事法规、航运术语等）
- 实时行业动态（运价、法规更新、市场趋势等）
- 产品功能介绍（视频监控、岸基值班、性能跟踪等）

## 工具

### smart_search
**统一信息检索入口**，根据depth参数自动选择搜索策略：

| depth | 搜索范围 | 适用场景 | 典型耗时 |
|-------|---------|---------|---------|
| quick | 术语速查→知识库FAQ→知识库Wiki | 平台概念、术语、简单FAQ | <3秒 |
| normal | 同quick + 官网站内搜索 + 增强网页搜索 | 一般航运/海事知识 | 5-15秒 |
| deep | 同normal + 多轮深度搜索(3-5组关键词) | 时效敏感、深度分析、行业动态 | 15-60秒 |

**depth选择规则**：
1. 用户问"是什么/怎么用/如何操作" → depth="quick"
2. 用户问"最新/最近/今天/今年/行业动态" → depth="normal"
3. 用户需要"分析/趋势/深度/全面/对比" → depth="deep"
4. 不确定时默认 depth="normal"

**示例调用**：
- smart_search(query="绿点是什么", depth="quick") → 术语直接命中
- smart_search(query="AIS信号不好", depth="normal") → 知识库+搜索
- smart_search(query="航运市场运价趋势", depth="deep") → 多轮深度搜索

## 搜索结果处理

### FAQ/标准回复匹配
当检索到FAQ条目时，严格按照标准回复内容回答，不添加额外信息。

### 主题说明
当仅匹配到主题说明时，可基于主题内容谨慎回答，但需注明是概述而非精确FAQ。

### 知识检索无结果
